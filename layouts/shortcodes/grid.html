Image grid

For creating an image grid the .gallery wrapper adds the object-fit and aspect-ratio for photos that vary in size and orientation, by default they use 16 / 9 defined by the variable [--aspect-ratio: 16 / 9].

The variable allows adjusting the ratio inline on the wrapper element, e.g. [style="--aspect-ratio: 3 / 4"].
